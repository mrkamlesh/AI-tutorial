# GenAI + RAG Tutorial Path (Frontend-Friendly)

This folder contains a practical learning path for a frontend developer who is starting in GenAI.

## How to use these tutorials

1. Read in order from `01` to `07`.
2. Build a tiny feature after each file (small wins).
3. Keep one notebook (`notes.md`) for:
   - New terms
   - API call examples
   - "What broke and why" notes

## Learning order

## Prerequisites

- Node.js 18+ (or Python 3.10+ if using Python examples)
- Basic REST API and async/await familiarity
- Azure account (for tutorial 03)

## Outcome after finishing

You should be able to design and ship a production-oriented RAG feature with:

- A clear pipeline design
- LLM integration via Azure OpenAI
- Vector search via Qdrant or Chroma
- Safety and guardrails
- Monitoring for quality and hallucinations
